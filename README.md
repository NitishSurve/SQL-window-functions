🧮 SQL Window Functions Practice

This repository contains practice problems and solutions for SQL Window Functions. It focuses on commonly used analytical functions that help perform calculations across rows related to the current row.

📚 Topics Covered

✨ ROW_NUMBER() – Assigns a unique number to each row within a partition
✨ RANK() – Ranks rows with possible gaps in ranking
✨ LAG() – Accesses data from the previous row
✨ LEAD() – Accesses data from the next row
✨ NTILE() – Divides rows into equal groups (buckets)

🎯 Purpose

🚀 Strengthen SQL analytical query skills
📊 Understand how window functions work with partitions and ordering
💡 Practice real-world data analysis scenarios
